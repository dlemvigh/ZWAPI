Introduction

[Description]

This package contains the z-wave api event class, and all z-wave api event listeners, used when receiving messages from the z-wave network.

Usage

[Description]

Because of the time delay between a request and response in any wireless network,and javas inefficiency when handling callback functions, all incomming z-wave messages is converted into z-wave events. The z-wave api will trigger an event based on the command class of the message, in the node object related to the node that sent the message. If for example a z-wave node with an id 1 sends a BASIC_SET from the command class COMMAND_CLASS_BASIC, The ZWNode object with id 1 will fire an event that will be picked up by the ZWEventListener, and the ZWBasicListener.
